# 📚 Online Bookstore Database Analysis | SQL Server

## 📌 Project Overview

This project focuses on designing and analyzing an Online Bookstore database using SQL Server.

The project includes database creation, table relationships, data insertion, and SQL queries to generate meaningful business insights related to books, customers, orders, and sales performance.

## 🛠️ Tools & Technologies Used

- SQL Server
- SQL Queries
- Database Design
- Joins
- Aggregate Functions
- Subqueries
- Data Analysis

## 🗄️ Database Features

The database contains information related to:

- 📖 Books
- 👥 Customers
- 🛒 Orders
- 💰 Sales Transactions

## 📊 Analysis Performed

The project covers:

- Total sales analysis
- Customer purchase behavior
- Best-selling books analysis
- Order and revenue tracking
- Category-wise book performance
- Customer and sales insights using SQL queries

## 🔍 SQL Concepts Demonstrated

- CREATE DATABASE
- CREATE TABLE
- INSERT Statements
- Primary Key & Foreign Key
- Constraints
- INNER JOIN / LEFT JOIN
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- Subqueries
- Filtering & Sorting Data

## 🎯 Business Objective

The objective of this project is to demonstrate how SQL can be used to manage bookstore data and extract valuable insights that help in understanding customer behavior and sales performance.
